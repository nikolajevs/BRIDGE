# Бридж — онлайн карточная игра

Карточная игра для 2–6 игроков за общим столом. Колода 36 карт (6 — туз).
Node.js + WebSocket, клиент на чистом HTML/CSS/JS — без сборки и без базы данных.

## Запуск

```bash
npm install
npm start          # http://localhost:3000
```

Порт можно поменять: `PORT=8080 npm start`.

Тесты:

```bash
npm test           # логика правил + 400 случайных партий
node server/index.js &   # сервер для сквозного теста
node test/e2e.js         # два клиента играют партию по WebSocket
```

## Как играть

1. Каждый открывает сайт, вводит имя.
2. Один создаёт стол и говорит остальным 4-значный код (или стол виден в лобби).
3. Создатель стола нажимает «Начать игру» (2–6 игроков).

Обрыв связи не выбрасывает из партии — при повторном открытии страницы
игрок возвращается на своё место (токен хранится в localStorage браузера).

## Аккаунты и топ-100

Играть можно **гостем** (просто ввести имя — как раньше) или через **аккаунт**
(логин + пароль). У аккаунта есть личный кабинет со статистикой (число партий,
побед, винрейт, выигранные раунды) и таблица **топ-100 игроков по числу побед**.
Пароли хранятся хешированными (scrypt). Данные — в SQLite (см. `DEPLOY.md`).

## Правила

- Раздача: всем по 5 карт, дилеру 4 — его «пятая» карта уходит вслепую первой
  на стол. Ходят по часовой стрелке.
- Ход: карта **той же масти** (любого достоинства) либо карта **того же
  достоинства** любой масти.
- Нечем ходить — берёшь 1 карту из прикупа; её можно сразу сыграть или спасовать.
- Прикуп кончился — сброс тасуется заново (кроме верхней карты).
- На ход даётся **30 секунд**. Не успел — наказание: сервер берёт за тебя карту из прикупа и пасует (ничего из руки не разыгрывается). В цепочке восьмёрок вместо этого берёшь накопленный штраф и пропускаешь ход; уже начатое накрытие шестёрки доигрывается. Это же спасает партию, если игрок отключился на своём ходу.

Особые карты:

| Карта | Эффект |
|---|---|
| 6 | Ходом нельзя закончить: игрок сам накрывает её (той же мастью любого достоинства, девяткой или валетом). Другой шестёркой — меняет масть, но ход не закрывает. Нечем — тянет из прикупа, пока не накроет. |
| 7 | Следующий берёт 1 карту и ходит. |
| 8 | Следующий берёт 2 карты и пропускает ход, но если у него есть 8 — обязан её положить: штраф растёт по +2 и переходит дальше. В цепочке восьмёрок 9 и валет не работают. |
| 9 | Универсальная: кладётся на любую карту. На девятку можно ответить девяткой (9 на 9). За один ход кладётся только одна девятка. |
| Валет | Кладётся на любую карту, игрок заказывает масть. Если победным ходом раунда стал валет — очки проигравших за раунд умножаются: свой один валет — ×2. Игрок может скинуть **все свои валеты за один ход**, если это его последние карты: два валета — ×3, три — ×4, и так далее. Валеты от разных игроков множитель не суммируют. |
| Дама пик | Следующий берёт 5 карт и пропускает ход. |
| Король | Четыре короля подряд — положивший четвёртого мгновенно выигрывает всю партию. |
| Туз | Следующий пропускает ход (отбиться нельзя). В игре 1×1 ход возвращается — тузы можно скидывать один за другим. |

Очки в конце раунда (за карты, оставшиеся на руках): 6, 7, 8, 9 — 0 очков;
туз — 15; остальные (10, валет, дама, король) — 10.
Пока у игрока счёт не открыт (0 очков), раунд записывается только если набрал
**30 и более**; меньше — не открывает счёт. После открытия счёта считаются все
раунды подряд. Ровно 125 — счёт обнуляется (и снова закрывается — до открытия
опять нужно 30+). Больше 125 — игрок выбывает. Играют до последнего.

## Решения по спорным местам

Эти детали не были жёстко оговорены — реализованы так (легко поменять в `server/game.js`):

- **После валета** подходит любая карта заказанной масти.
- **Слепая шестёрка** при раздаче: её накрывает сам дилер, потом ход идёт дальше.
  Остальные слепые эффекты (7/8/дама пик/туз) применяются к первому ходящему.
- **Конец раунда с висящим штрафом** (последней картой вышла 8 или дама пик):
  следующий игрок добирает штрафные карты, и только потом считаются очки.
- **Добор**: взять карту из прикупа можно даже при наличии хода; после добора
  можно сыграть любую подходящую карту (не обязательно взятую) или спасовать.
- **Игрок покинул игру** — выбывает, его карты выходят из оборота до конца раунда.

## Структура

```
server/index.js   HTTP-статика + WebSocket: лобби, столы, протокол
server/game.js    вся игровая логика (чистый класс, без сети)
public/           клиент: index.html, style.css, client.js
test/smoke.js     тесты правил + случайные партии с инвариантами
test/e2e.js       сквозной тест по WebSocket
```

## Деплой

Любая машина с Node.js ≥ 18: `npm install && PORT=80 node server/index.js`.
За обратным прокси (nginx/caddy) нужно проксировать и WebSocket-апгрейд.
Состояние хранится в памяти процесса — перезапуск сервера сбрасывает столы.

Автоматический деплой на VPS при каждом push в `main` — см. **[DEPLOY.md](DEPLOY.md)**.

## Android-приложение

В папке **[android/](android/)** — тонкая WebView-обёртка (Android Studio проект):
полноэкранное приложение, открывающее сервер как обычную игру. Обновления игры
не требуют пересборки APK. Как указать адрес сервера и собрать APK — в
[android/README.md](android/README.md).
